# Habit Tracker

Full-stack website built with Java Spring Boot, Thymeleaf, and CSS, that allows users to track their daily habits and see their progress. 

## Features
- Add, delete, and view habits
- Track daily completion and streaks
- Dashboard overview of progress

## Tech Stack
- Java 21
- Spring Boot
- Thymeleaf (HTML templates)
- CSS
- H2 Database
